LectorCSV es un breve programa ejecutado en Terminal que su función es, mediante un archivo .csv que el usuario elije, creará un objeto (siguiendo un formato específico [ID,PRODUCTO,PRECIO,STOCK]) y posteriormente creará el directorio "informes" y dentro generará un archivo .txt donde aparecerán los datos del propio CSV más el número de productos total y la media de los precios. En la propia carpeta se deja el fichero productos.csv que sirve de ejemplo de lo que espera el programa para poder funcionar.
